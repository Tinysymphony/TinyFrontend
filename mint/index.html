About TinyMint

TinyMint is a curriculum work of ZJU B/S Web Design. TinyMint is designed to help users to create their own responsive webpages by integrated templates. Users can easily design and download HTML/CSS/JavaScript from the online webpage editor. Mint is inspired by MS Sway and is trying to provide users a better UE. The Mint home page is using full screen layout from Codrops.
